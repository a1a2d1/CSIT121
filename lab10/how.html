How is it used?

Using an EPUB file is pretty straight forward. An app or E-reader is required. For Apple iPhone/iPad, the Apple Books app can be used. On Android devices, Google Play Books can be used (Continual Engine, n.d.; ). Some popular e-readers are Amazon Kindle, Barnes & Noble Nook, and Kobo e-readers.

An EPUB file is esentially a ZIP archive with a predefined structure that uses HTML, CSS, XML, and optional video or audio assets. XHTML/HTML files are the chapters. CSS is used for styling. OPF file is used for metadata. A navigation file is used for the table of contents, and images, fonts, audio files, and video are assets.

EPUB Example

HTML


            <?xml version="1.0" encoding="utf-8"?>
            <html xmlns="http://www.w3.org/1999/xhtml">
            <head>
              <title>Chapter 1</title>
              <link rel="stylesheet" href="styles.css"/>
            </head>
            <body>
              <h1>Chapter 1</h1>
              <p>Chapter 1 contents...</p>
            </body>
            </html>

CSS


            body {
              font-family: serif;
              line-height: 1.5;
            }

            h1 {
              text-align: center;
            }

container.xml


          <?xml version="1.0"?>
          <container version="1.0" xmlns="urn:oasis:names:tc:opendocument:xmlns:container">
            <rootfiles>
                <rootfile full-path="content.opf"
                media-type="application/oebps-package+xml"/>
            </rootfiles>
          </container>
