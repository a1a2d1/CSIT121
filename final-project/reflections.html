My Reflection

This project helped me understand how important it is to plan upfront for any non trivial website. Early in the project, I started building pages without planning and ended up having to redo a good part of this project. Planning ahead with layouts, content structure, and styling decisions can save a lot of time and make the development process much smoother.

This course as a whole was very informative. I learned a lot and more importantly, learned how much I don't know and how much more there is to learn. One of the biggest challenges I faced was this semester was working with CSS layouts. At first, it was difficult to understand how different layout techniques worked together, especially since I still remember doing layouts with tables back in the day. I feel much more comfortable with CSS now than I did at the beginning of the semester.

Another major takeaway from the course was learning about accessibility. Before taking this class, I did not fully understand how important accessibility is for web development. I learned that websites should be designed so that all users, including people with disabilities, can navigate and use them effectively.

Overall, this course taught me a lot about both the technical and creative sides of web development. I improved my understanding of HTML and CSS, gained experience creating responsive layouts, and learned the importance of planning, testing, and accessibility.
