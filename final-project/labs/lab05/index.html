Reflection After Lab-05

Layout in HTML and CSS can definitely be tricky. This lab took some time to complete. These are a few tips on best practices for designing the layout. Use Flexbox for navigation menu, CSS cards, simple rows or columns. Use CSS Grid for the page layout, image galleries, or when precise alignment is needed. Use a mobile first approach. Mobile layout is simpler. Mobile phones have less resources. It's best to design for mobile first.
